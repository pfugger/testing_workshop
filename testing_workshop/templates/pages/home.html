This project has been written for a workshop on testing, covering how to write tests and code together, how to run tests and how to automate test running on collaborative projects.

The finished project allows the user to retrieve a transcription of a letter from an external website, automatically generate named entities for tokens in the transcription, and display the named entities alongside an image of the original letter page.

To get started go to Named Entity Recognition
